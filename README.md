
# Sales Analysis Dashboard - Power BI

## Overview
This Power BI dashboard provides interactive visualizations and insights into sales performance, profit trends, customer purchasing behavior, and payment methods. The dashboard helps in understanding business growth and analyzing key sales metrics effectively.

## Features
- Sales and Profit Analysis
- Category-wise Sales Distribution
- Payment Mode Insights
- Monthly Profit Trend Analysis
- Interactive Charts and Visualizations

## Tools & Technologies Used
- Power BI
- Excel
- DAX
- Data Visualization

## Key Insights
- Clothing category contributed higher sales compared to other categories.
- COD was identified as one of the most preferred payment methods.
- Monthly profit trends helped analyze business performance and growth patterns.
